Crea una clase llamada Producto que tenga 4 atributos de tipos diferentes:

un String para el nombre,

un double para el precio,

un int para el stock,

un boolean para indicar si está en oferta.

La clase debe tener tres métodos:

  a) mostrarInfo(): muestra por pantalla los valores de los atributos.

  b) aplicarDescuento(): si el producto está en oferta, reduce su precio un 10%.

  c) main(): crea un array de 4 productos, pide los datos de 4 productos por teclado, aplica descuentos cuando corresponda y muestra la información.
